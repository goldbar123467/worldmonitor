# The Great American Road Trip — Summer Project

A static one-page site for a 5th & 6th grade summer geography project. Students plan a 7–14 day U.S. road trip on a $2,000 budget, then build the result as a Canva slideshow.

## Files

- `index.html` — the page
- `styles.css` — comic-sticker / beach theme
- `script.js` — region-roulette button

## Run locally

Open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

Then visit http://localhost:8000.
